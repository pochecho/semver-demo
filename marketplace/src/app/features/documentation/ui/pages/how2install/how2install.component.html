Cómo instalar la librería del Sistema de Diseño


En esta sección te daremos una guía práctica, referente a como instalar la librería. Lo primero a tener presente es que tipo de tecnología estas empleando sea web o mobile. A continuación expondremos como puedes proceder en cada uno de los escenarios.


Desarrolladores web en angular

Primero debes crear en la raíz del proyecto del archivo .npmrc y en el archivo hacer referencia al registry de Bancolombia

registry=https://artifactory.apps.bancolombia.com/api/npm/npm-bancolombia/

luego debemos agregar la dependencia mediante npm

npm install @bancolombia/design-system-web.

En el archivo angular.json agregar las siguiente configuración en las secciones respectivas: 

                            
    "styles" : [
                "src/styles.css",
                "node_modules/@bancolombia/design-system-web/bds.min.css"
                ],
    "scripts": [
                "node_modules/@bancolombia/design-system-web/bds.min.js"
                ]

Esta Librería cuenta con distintos módulos, a medida que el proyecto requiera debe hacer las importaciones respectivas en el módulo de angular que lo necesite. En el catalogo de componentes puede encontrar mas información para implementar el elemento que desees.



Desarrolladores web en otros frameworks

En Bancolombia, nuestra herramienta principal de desarrolo web es Angular. Sin embargo, algunos proyectos pueden necesitar que su solución sea desarrollada en otra tecnología de desarrollo web. Si este es tu escenario, a continuación expondremos como podemos instalar nuestra librería en una plantilla HTML normal.


Primer paso: Importar minificados de estilos

En la cabercera de tu html te debes traer la versión del sistema de diseño que desees utilizar, de la siguiente forma

                        
    <‍head> 
        ...
        <‍link rel="stylesheet" href="https://library-sdb.apps.bancolombia.com/bds/2.1.2/bds.min.css" />
    <‍/head>

Si su componente no requiere de comportamientos y lo que necesitas es únicamente los estilos, con la configuración anterior ya tienes todo listo. Recomendamos fuertemente apuntar a una versión específica del BDS.


Segundo paso: Importar minificados de comportamientos

En una etiqueta tipo script o según la configuración del framework debe importarse la versión específica del minificado JavaScript.

                        
    <‍script src="https://library-sdb.apps.bancolombia.com/2.0.20/js/bds.min.js"><‍/script>

Esta Librería cuenta con distintos métodos, que debe ejecutar para la correctar configuración de sus componentes. En el catalogo de componentes puede encontrar mas información para implementar el elemento que necesite.



Desarrolladores Flutter

En Bancolombia, nuestra herramienta principal de desarrolo mobile es Flutter. Para instalar la librería en Flutter debes seguir los siguientes pasos.


Primer paso: Importar librería

En en tu pubspec.yaml debes importar la librería de la siguiente forma:

                        
    provider: ^6.0.1
    bancolombia_foundations:
      git:
        url: https://grupobancolombia.visualstudio.com/Vicepresidencia%20Servicios%20de%20Tecnolog%C3%ADa/_git/NU0066001_BDS_MOBILE_MR
        path: bancolombia_foundations/
        ref: 3.2.0
    bds_mobile:
      git:
        url: https://grupobancolombia.visualstudio.com/Vicepresidencia%20Servicios%20de%20Tecnolog%C3%ADa/_git/NU0066001_BDS_MOBILE_MR
        path: bds_mobile/
        ref: 3.2.0

Cuando artifactory sea compatible con Flutter podrás tener esta instalación utilizando la forma tradicional. Por ahora debes apuntar al repo y al versión específica que deseas.


Segundo paso: Configura el provider

Dado que tenemos soporte a multimarca en el MultiProvider de tu app debes especificar que tema vas a utilizar como se ve en el siquiente ejemplo :

                        
    return MultiProvider(
        providers: [
            ...BancolombiaFoundations.themeProvider
        ]

Con esto ya está listo para utilizar la librería de Flutter.


Para ver el detalle de como utilizar algún componente puede dirigirse all catalogo de componentes donde encontrar mas información.
